# SlimEmail

Khi user nhắc SlimEmail / email marketing / list / campaign / brand APLUS:

1. Đọc `.claude/skills/slimemail-ai-agent/SKILL.md` (và `USECASES.md` nếu cần prompt mẫu).
2. Dùng biến `SLIMEMAIL_*` từ `.env` — không hỏi lại base URL/key nếu đã có.
3. Luôn `GET /me` trước; xóa/gửi/merge: `dry_run` → user OK → `confirm: true`.
4. Trả lời tiếng Việt.
